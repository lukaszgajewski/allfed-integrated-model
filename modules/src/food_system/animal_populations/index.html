Class to store animal population data in. Needs to store the following: animal type, population, and slaughter.

Parameters

animal_type : str Type of animal (beef, pork, chicken etc...) population : int Number of animals (total) slaughter : int Number of animals slaughtered this month pregnant : int Number of animals pregnant this month starving : int Number of animals starving this month feed_required : int Amount of feed required this month
